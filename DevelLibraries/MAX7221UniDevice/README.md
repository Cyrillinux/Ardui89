MAX7221UniDevice
==========
MAX7221UniDevice is an ARDUINO https://sourceforge.net/project/max7221-arduino library for MAX7219 and MAX7221 Led display drivers for 8051 and AVR architectures
The code also works with the https://sourceforge.net/project/max7221-arduino link.

Documentation
-------------
Documentation for the library is on the https://sourceforge.net/project/max7221-arduino link.

Download
--------
The lastest binary version of the Library is always available from the https://sourceforge.net/project/max7221-arduino link.

Install
-------
The library can be installed using the [standard Arduino library install procedure](http://arduino.cc/en/Guide/Libraries)
